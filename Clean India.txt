CLEAN INDIA
One QR. One Community. One Clean Corner at a Time.

CORE IDEA

Clean India is a minimalist, QR-first civic coordination app that enables Indian gallis and housing societies to organize and execute local clean-ups within 24 hours — without depending on government systems, without complex technology, and without handling money inside the app.

It is not a complaint app.
It is not a reporting tool.
It is not a payment gateway.
It is not social media.

It is a coordination scaffold for real-world action.

The app turns one physical location (a gate, lane, society entrance) into a digital action node through a permanent QR code.

PROBLEM STATEMENT

Indian lanes and societies often face:

Garbage accumulation in narrow lanes

Result:
Normalized filth.

USER ROLES

There are two global identity states:

A. Regular User
B. Volunteer / Clean Worker

Inside a society, roles are:

Admin (Main Admin)

Co-Admin / Sub-Admin

Member

Important:
Any individual can join maximum 2 societies total.
Role does not override this limit.

Examples:

Admin in Home Society + Member in Office Society

Co-Admin in Society A + Member in Society B

Member in 2 societies
Never more than 2.

SOCIETY CREATION

Flow:

Tap "Create Society"

Enter society/galli name (duplicates allowed globally)

System generates permanent unique slug:
Example: cleanindia.app/jX9k2

Permanent QR is generated

Creator becomes Admin #1

There is:

No name validation

No uniqueness enforcement on society name

Only slug is unique

QR becomes the digital identity of that community.

JOINING A SOCIETY

Flow:

Scan QR OR enter slug manually

Enter flat/house number (e.g., B-304)

Join instantly

No:

OTP

Admin approval

Verification

Waiting

Hard Rule:
User cannot join more than 2 societies.
Enforced at Firestore security rule level.

Feed displays posts only from joined societies.

CLEAN-UP POOLING SYSTEM

When issue escalates:

Admin taps "Start Clean-Up"

App:

Overlays society name on admin’s pre-saved UPI QR

Posts in society feed

Members:

Pay directly to admin’s UPI

₹1–₹3 micro contributions

App handles ZERO money.

Admin:

Hires worker OR organizes volunteers

Posts before/after images

GOVERNANCE SYSTEM

Admin Powers:

Start clean-up

Add Co-Admin

Remove misbehaving members

Transfer admin role

Co-Admin Powers:

Remove members

Assist governance

Member:

Post issues

Chat (3/day)

Leave society

Admin Rotation Rule:
Every month:

Main admin must change OR

Election must occur

Purpose:

Distribute responsibility

Prevent power concentration

Build civic participation

If admin refuses:
Members can fork.

FORKING (SELF-CORRECTION)

If trust breaks:

Members leave society

Create new society (same name allowed)

New QR generated

Share new QR

Community migrates

No bans.
No global punishment.
Social reality corrects leadership.
